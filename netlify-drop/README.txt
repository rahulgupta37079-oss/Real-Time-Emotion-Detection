PassionBots - Real-Time Emotion Detection
==========================================

NETLIFY DROP DEPLOYMENT PACKAGE
================================

This folder contains everything you need to deploy PassionBots Emotion Detection to Netlify Drop.

INCLUDED FILES:
- index.html (Complete standalone application)
- app.js (Emotion detection JavaScript logic)
- _redirects (Netlify routing configuration)
- README.txt (This file)

HOW TO DEPLOY TO NETLIFY DROP:
===============================

1. Go to: https://app.netlify.com/drop

2. Drag and drop this ENTIRE FOLDER (netlify-drop) to the Netlify Drop zone

3. Wait for deployment to complete (usually 10-30 seconds)

4. Your site will be live at: https://[random-id].netlify.app

5. (Optional) Customize your URL in Netlify dashboard under Site Settings > Domain Management

FEATURES:
=========
✅ Real-time emotion detection with face-api.js (TensorFlow.js)
✅ 7 emotion categories: Neutral, Happy, Sad, Angry, Fearful, Disgusted, Surprised
✅ Multi-face detection capability
✅ Anger alert system with configurable thresholds
✅ Live audio visualization
✅ Professional black/yellow/white UI design
✅ 100% client-side processing (privacy-first)
✅ Mobile responsive
